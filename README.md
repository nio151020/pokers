# 德州扑克胜率计算器

一个基于蒙特卡洛模拟的德州扑克胜率计算工具，支持 3-5 人对局。

## 功能特点

- 🎲 **蒙特卡洛模拟**：可调整模拟次数，快速估算胜率
- 🎴 **可视化牌盘**：点击选择手牌和公共牌
- 👥 **3-5 人支持**：可设置玩家数量，支持弃牌标记
- 📊 **胜率历史**：记录每个阶段的胜率变化
- 📱 **响应式设计**：适配手机和桌面

## 使用方法

1. 选择你的手牌（点击"我"的位置）
2. 点击阶段按钮（翻牌前/翻牌/转牌/河牌）
3. 根据阶段翻开相应数量的公共牌
4. 点击"弃牌"按钮标记已弃牌的对手
5. 实时查看胜率变化

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需后端，完全前端计算
